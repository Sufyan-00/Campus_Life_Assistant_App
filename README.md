# Campus Life Assistant App

**Campus Life Assistant App** is a comprehensive mobile application designed to help students manage their schedules, assignments,and feedback collection seamlessly. The app is developed using **FlutterFlow** and **Firebase**.

## Developed by:
- **Sufyan Abdullah** - SP22-BSE-025

---

## Features Overview

### 1. **User Authentication**
- Firebase Authentication for secure sign-up, login, and profile management.
- Password recovery functionality for user convenience.

### 2. **Class Schedule Management**
- Add, edit, and delete class schedules.
- Synchronize schedules across devices using Firebase Firestore.

### 3. **Event Notifications [feature to be added in future]**
- Local notifications for upcoming classes, assignments, and important deadlines.
- Push notifications using Firebase Cloud Messaging (FCM).

### 4. **Assignment Tracker**
- Add and manage assignments with deadlines.
- Receive reminders via local notifications before submission deadlines.
- Sync assignments with Firebase Firestore for cloud access.

### 5. **Study Group Finder [feature to be added in future]**
- Create, join, or leave study groups.
- Store and retrieve study group data using Firebase Firestore or Realtime Database.

### 6. **Feedback System**
- Submit feedback on courses, professors, and campus services.
- Store feedback in Firebase Firestore.

### 7. **Offline Access [feature to be added in future]**
- Utilize SQFlite for local data storage when offline.
- Synchronize data with Firebase once the user is back online.

### 8. **UI Enhancements and Final Integration**
- A user-friendly, polished UI designed for intuitive navigation.
- All features seamlessly integrated and tested for smooth performance.

---

## Setup and Installation

### Prerequisites
1. Install [Flutter](https://flutter.dev/docs/get-started/install).
2. Install [FlutterFlow](https://flutterflow.io).
3. Set up a Firebase project with the following enabled:
   - Firebase Authentication
   - Firestore Database
